=== HOW TO USE ===

STEP 1: Generate Icons & Load Extension
   - Open generate-icons.html → Generate icons
   - Load extension in chrome://extensions/

STEP 2: Configure Translator
   - Click extension icon
   - Go to "Translator Services" section
   - Enable the translators you want to use
   - Google Translate is enabled by default

STEP 3: Set Display Mode
   - In "Translation Settings"
   - Select "Inline in Meet (Best)" for best experience
   - This shows translation directly in the meeting

STEP 4: Start Translating
   - Join Google Meet
   - Enable captions (CC button)
   - Click extension icon → Start
   - Speak or play audio
   - See translation appear below video!

Built-in Services:
✓ Google Translate (Free) - Enabled by default
✓ LibreTranslate (Free, Open Source) - Enable in popup
✓ MyMemory (Free) - Enable in popup

The extension tries enabled translators in order:
1. First enabled translator → If fails →
2. Second enabled translator → If fails →
3. Falls back to Google Translate URL

Example 2: DeepL API (Requires API Key)

NEW Way (Inline Mode):

=== PRIVACY & SECURITY ===

What gets sent to translators:
✓ Only caption text (that you enabled in Meet)
✓ Source and target language codes

What does NOT get sent:
✗ Audio or video
✗ Meeting URLs or IDs
✗ Participant information
✗ Any other meeting data

Default translators (built-in):
- Google Translate: Reputable, widely used
- LibreTranslate: Open source, can self-host
- MyMemory: Free public service

Custom translators:
- Check the privacy policy of the service
- Only use trusted services
- For sensitive meetings, use self-hosted LibreTranslate

=== TROUBLESHOOTING NEW FEATURES ===

Inline translation not showing:
1. Make sure "Inline in Meet" is selected
2. Check browser console (F12) for errors
3. Verify captions are enabled in Meet
4. Try refreshing the Meet page

Translator not working:
1. Check if translator is enabled (checkbox)
2. Open console and look for fetch errors
3. Some APIs require CORS support
4. Try disabling and using only Google Translate
5. Check translator-examples.txt for correct format

Translation quality issues:
1. Try different translator service
2. DeepL has best quality (but requires API key)
3. Google Translate is most reliable
4. LibreTranslate is good for privacy

=== ADVANCED: SELF-HOSTED TRANSLATION ===

For maximum privacy, run your own LibreTranslate:

1. Install: pip install libretranslate
2. Run: libretranslate --host 0.0.0.0
3. In extension, disable all translators
4. Add custom translator:
   Name: My LibreTranslate
   Type: libretranslate
   URL: http://localhost:5000/translate
   Method: POST
5. Enable it

Now all translations stay on your computer!
